Опыт работы с технологиями

В проекте я использовала следующие технологии:

  • Java – для разработки серверной логики и обработки данных.
  • SQL – для работы с базой данных, написания запросов и оптимизации.
  • JPA и Hibernate – для объектно-ориентированного взаимодействия с базой данных, автоматического преобразования объектов в SQL и управления транзакциями.
  • RESTful API – для создания веб-сервисов и взаимодействия с клиентами через HTTP-запросы.
  • MySQL – в качестве базы данных для хранения данных проекта.
  • Spring Boot – для быстрой разработки серверной части, реализации RESTful сервисов и управления зависимостями.

Эти технологии обеспечили эффективную и удобную реализацию всех функциональных возможностей проекта.
